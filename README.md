# Arithmetic_Quiz_CLI
 Your Terminal Math Trainer
    Ditch the flashcards. Level up your math game without ever leaving your terminal! This CLI tool is a fast, fun way to drill arithmetic. Just run the program, answer the question, and watch your mental speed soar. 
    Randomly generated problems across four operations.
    
    Features:
    
    Real-time scoring and progress tracking.

    Instant feedback on correct and incorrect answers.

    Lightweight, fast, and terminal-native.
